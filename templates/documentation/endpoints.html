API Endpoints

Introduction

The API is short and sweet, so let's get started.

Base URL

https://scp-api.adxxmm.repl.co/api/

/scps

This endpoint allows you to get data about an SCP by its number.

Example

https://scp-api.adxxmm.repl.co/api/scps/999/

Returns

{"found": (boolean), "scp_info": {"item_number": "", "object_class": "", "description": ""}})
