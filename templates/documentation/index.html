The SCP API

Welcome to the SCP API Documentation! This project was created to provide information about every SCP through a friendly and simple API!

Created by Adxm!

Why?

It has been traditionally difficult to access the SCP wiki programmatically without the use of web scraping. This API allows users who may not know how to, or want to avoid web scraping, access information about SCPs!
